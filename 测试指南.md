# 🤖 双足机器人完整测试指南

> 发给现场测试人员的操作手册。包含舵机测试、IMU测试、和闭环步行。

---

## 准备工作

### 硬件清单
- [ ] 双足机器人下半身 (10个舵机, ID 1~10)
- [ ] YIS321 IMU 模块
- [ ] 7.4V 2S 锂电池 (充满电)
- [ ] **USB 转串口模块 ×2** (一个接舵机, 一个接 IMU)
- [ ] USB 线 ×2

### 接线图
```
电脑
├── USB1 → 串口模块A (COM3) → 舵机总线信号线
│                              └→ 电池 7.4V 供电
└── USB2 → 串口模块B (COM5) → YIS321 IMU
                               └→ 3.3V/5V 供电
⚠️ 所有 GND 必须共地!
```

### 软件准备
```bash
pip install pyserial
```

### 确认串口号
打开 **设备管理器 → 端口(COM和LPT)**，记下两个串口号。

## 第三部分: 闭环步行 (舵机+IMU)

> ⚠️ **必须先通过第一和第二部分测试!**

```bash
# 先试静态平衡 (站立 + 推它看反应)
python tools/walk_imu.py --servo-port COM3 --imu-port COM5 --mode balance

# 再试闭环步行
python tools/walk_imu.py --servo-port COM3 --imu-port COM5 --mode walk
```

| 模式 | 说明 |
|------|------|
| `stand` | 站立 + 显示 IMU 数据 |
| `balance` | 静态平衡 (UVC修正, 不迈步) — **先测这个!** |
| `walk` | UVC闭环步行 |

## ⚠️ 安全须知

1. **全程用手扶住机器人**
2. **异常立刻 Ctrl+C**
3. **电池 <6.5V 停止测试**
4. **舵机发烫暂停5分钟**
5. 闭环步行有跌倒检测 (>20° 自动停止)

## 调参速查

### 开环步行 (无需IMU)
```bash
python tools/walk.py --port COM3 --stride 15 --lift 10 --sway 5 --period 1.5
```

### 闭环步行
```bash
python tools/walk_imu.py --servo-port COM3 --imu-port COM5 --steps 10 --period 1.0
```

| 参数 | 含义 | 范围 |
|------|------|------|
| `stride` | 步幅 mm | 5~40 |
| `lift` | 抬脚高度 mm | 5~30 |
| `sway` | 侧摆 mm | 3~15 |
| `period` | 步态周期 s | 0.8~2.0 |
| `steps` | 步数 | 2~20 |
